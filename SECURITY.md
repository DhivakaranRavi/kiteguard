# Security Policy

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

kiteguard is a security tool — responsible disclosure is critical.

Use **[GitHub Private Security Advisories](https://github.com/DhivakaranRavi/kiteguard/security/advisories/new)** to report vulnerabilities privately. GitHub keeps the report confidential until a fix is released.

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Your suggested fix (optional)

We respond within **48 hours** and aim to ship a fix within 7 days for critical issues.

## Supported versions

| Version | Supported |
|---|---|
| Latest release | ✅ |
| Older releases | ❌ |

Always run the latest version.
